- To add more than one space between tho words, insert the new space using the special marker "&"
Example: Hello           World.

- To add a comment in brackets, use the following, removing both asterisk (*):
"<*!-- --*>"
-> Don't use this kind of comment to note project standarts. Users are able to read this by acessing the font code of the page.

- It's possible to add special characters to the text by using the special marker "&", same as I used to add more than one space between two words.
Examples:
-> Euro Symbol: € (& + euro) -> Pound: £ (& + pound) -> Yen: ¥ (& + Yen)
